Size¶

class Size[source]¶

Bases: qiskit.transpiler.basepasses.AnalysisPass

An analysis pass for calculating the size of a DAG circuit.

Attributes Summary

is_analysis_pass If the pass is an AnalysisPass, that means that the pass can analyze the DAG and write the results of that analysis in the property set.
is_transformation_pass If the pass is a TransformationPass, that means that the pass can manipulate the DAG, but cannot modify the property set (but it can be read).

Methods Summary

name() The name of the pass.
run(dag) Run a pass on the DAGCircuit.

Attributes Documentation

is_analysis_pass¶

If the pass is an AnalysisPass, that means that the pass can analyze the DAG and write the results of that analysis in the property set. Modifications on the DAG are not allowed by this kind of pass.

is_transformation_pass¶

If the pass is a TransformationPass, that means that the pass can manipulate the DAG, but cannot modify the property set (but it can be read).

Methods Documentation

name()¶

The name of the pass.

run(dag)[source]¶

Run a pass on the DAGCircuit. This is implemented by the pass developer. :param dag: the dag on which the pass is run. :type dag: DAGCircuit

Raises:NotImplementedError – when this is left unimplemented for a pass.
